# Light/Dark Mode dan Loading Screen

Folder `fix sqlite` sekarang punya light mode, dark mode, dan loading screen seperti pola yang ada di `p9`.

Perbedaannya: `fix sqlite` tetap tidak memakai AsyncStorage. Pilihan tema disimpan di SQLite.

## Apa Yang Ditambahkan

Fitur baru:

- tema `light`
- tema `dark`
- tombol ganti tema di kanan header
- loading screen saat data SQLite sedang disiapkan
- penyimpanan pilihan tema ke SQLite

## File Yang Berubah

Tema didefinisikan di:

```text
theme/colors.js
```

Di file itu ada:

```js
export const appThemes = {
  light: { ... },
  dark: { ... },
};
```

`App.js` memilih tema aktif lewat state:

```js
const [themeMode, setThemeMode] = useState("light");
```

Lalu warna tema dikirim ke screen:

```js
colors={colors}
```

Jadi screen seperti `Beranda`, `AdminProduk`, `Pesanan`, `Selesai`, `FormPesanan`, dan `ItemPesanan` bisa mengikuti warna tema aktif.

## Loading Screen

Sebelumnya `fix sqlite` langsung menampilkan navigator walaupun data SQLite belum selesai dibaca.

Sekarang ada state:

```js
const [isHydrated, setIsHydrated] = useState(false);
```

Saat app dibuka, `App.js` menjalankan:

```js
await initializeDatabase();
fetchProducts();
fetchOrders();
getSetting("themeMode");
```

Selama proses itu belum selesai, app menampilkan:

```text
Memuat aplikasi...
```

Ini mengikuti konsep loading screen di `p9`: UI utama baru muncul setelah data awal siap.

## Penyimpanan Tema

Di `p9`, tema disimpan memakai AsyncStorage.

Di `fix sqlite`, tema disimpan ke SQLite supaya project ini tetap bersih dari AsyncStorage.

Tabel baru di SQLite:

```sql
CREATE TABLE IF NOT EXISTS settings (
  key TEXT PRIMARY KEY NOT NULL,
  value TEXT NOT NULL
);
```

Key yang dipakai:

```text
themeMode
```

Nilainya:

```text
light
```

atau:

```text
dark
```

Saat tombol tema ditekan, app menjalankan:

```js
setSetting("themeMode", nextThemeMode);
```

Saat app dibuka lagi, app membaca:

```js
getSetting("themeMode");
```

Jadi pilihan tema terakhir tetap tersimpan.

## Perbedaan Dengan P9

`p9` memakai campuran:

```text
SQLite untuk produk/pesanan
AsyncStorage untuk tema dan migrasi data lama
```

`fix sqlite` sekarang memakai:

```text
SQLite untuk produk/pesanan/tema
```

Jadi `fix sqlite` tetap lebih konsisten dari sisi storage.

## Catatan Penting

SQLite tetap database lokal di masing-masing HP.

Light/dark mode ini hanya mengubah tampilan di HP itu sendiri. Kalau app dipasang di dua HP berbeda, pilihan tema dan data pesanan tetap berdiri sendiri di masing-masing HP.
