# Perbaikan Masalah White Screen Setelah Pemuatan Aplikasi

Dokumen ini menjelaskan penyebab masalah layar putih (*white screen*) setelah layar loading selesai, dan bagaimana masalah tersebut diperbaiki pada folder `fix sqlite`.

## Penyebab Error

Aplikasi mengalami crash di tingkat runtime dengan pesan kesalahan berikut:
```text
TypeError: Cannot read properties of undefined (reading 'bold')
    in HeaderTitle (at Header.tsx)
    in Header (at HeaderContainer.tsx)
    ...
```

Error ini disebabkan oleh ketidakcocokan kustomisasi tema di `App.js` dengan persyaratan baru dari **React Navigation v7**. 

Pada React Navigation v7:
1. Struktur objek tema memerlukan properti `fonts` (seperti `regular`, `medium`, `bold`, `heavy`).
2. Custom theme lama di proyek hanya mendefinisikan objek `colors` dan `dark`.
3. Akibatnya, saat React Navigation mencoba merender judul header melalui komponen `HeaderTitle`, aplikasi mencari nilai font bold (`theme.fonts.bold`) yang bernilai `undefined`, memicu `TypeError` dan mengakibatkan layar putih (*white screen*).

## Solusi / Perbaikan

Kita memperbaiki pembuatan objek `navigationTheme` di `App.js` agar mewarisi (*inherit*) properti default dari `DefaultTheme` (untuk mode *light*) dan `DarkTheme` (untuk mode *dark*) yang disediakan oleh `@react-navigation/native`.

Dengan cara ini, konfigurasi esensial seperti `fonts` bawaan platform tetap ada, sementara nilai `colors` kustom kita tetap diaplikasikan.

### Perubahan Kode di `App.js`

Berikut adalah perubahan yang dilakukan pada file [App.js](file:///d:/Repo/hivicode/mobile-4/fix%20sqlite/App.js):

```diff
-import { NavigationContainer } from "@react-navigation/native";
+import { NavigationContainer, DefaultTheme, DarkTheme } from "@react-navigation/native";

...

-const navigationTheme = (colors, themeMode) => ({
-  dark: themeMode === "dark",
-  colors: {
-    primary: colors.primary,
-    background: colors.background,
-    card: colors.card,
-    text: colors.text,
-    border: colors.border,
-    notification: colors.secondary,
-  },
-});
+const navigationTheme = (colors, themeMode) => {
+  const baseTheme = themeMode === "dark" ? DarkTheme : DefaultTheme;
+  return {
+    ...baseTheme,
+    dark: themeMode === "dark",
+    colors: {
+      ...baseTheme.colors,
+      primary: colors.primary,
+      background: colors.background,
+      card: colors.card,
+      text: colors.text,
+      border: colors.border,
+      notification: colors.secondary,
+    },
+  };
+};
```

## Hasil Setelah Perbaikan

Setelah perbaikan ini diterapkan:
1. Layar loading "Memuat aplikasi..." berjalan terlebih dahulu saat data SQLite sedang diinisialisasi.
2. Setelah inisialisasi database selesai, aplikasi berhasil memuat halaman navigasi utama tanpa mengalami crash.
3. Light dan Dark mode berjalan normal dengan warna status bar, header, dan tab navigation yang mengikuti tema kustom secara konsisten.
